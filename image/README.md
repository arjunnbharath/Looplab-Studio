# Image assets

| Folder | Use |
|--------|-----|
| **home/** | Homepage hero: `background section1.png`, `sec1pic.png`–`sec4pic.png` |
| **products/** | Shared catalog grid: `1.png`–`11.png`, `j.png` |
| **women/** | Optional: women-only shots (pages currently use `products/` + `home/`) |
| **men/** | Optional: men-only shots |
| **kids/** | Optional: kids-only shots |

Paths in HTML/JS use forward slashes, e.g. `image/products/1.png`, `image/home/sec1pic.png`. Spaces in filenames stay encoded as `%20` in `src` URLs where needed.
