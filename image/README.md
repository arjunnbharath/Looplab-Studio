# Image assets

| Folder | Use |
|--------|-----|
| **home/** | Homepage hero: `background section1.png`, `sec1pic.png`–`sec4pic.png` |
| **products/** | Shared catalog grid: `1.png`–`11.png`, `j.png` |
| **new/** | **new.html** hero Swiper + “Shop the drop” grid: `1.png`–`12.png` |
| **women/** | **women.html** product grid only: `1.png`–`12.png` (replace with your women shots) |
| **men/** | **men.html** product grid only: `1.png`–`12.png` |
| **kids/** | **kids.html** product grid only: `1.png`–`10.png` |

Paths in HTML/JS use forward slashes, e.g. `image/products/1.png`, `image/new/1.png`, `image/home/sec1pic.png`. Spaces in filenames stay encoded as `%20` in `src` URLs where needed.
