# Complimentary gift image

The cart uses **`gift.jpg`** in this folder for the Mother’s Day complimentary gift line.

- Add your photo as: **`gift.jpg`** (same folder as this file).
- Or use another filename and update **`GIFT_IMAGE_URL`** in `js/page-cart-page.js` (near the top, next to `GIFT_LINE_ID`).

Supported formats: `.jpg` / `.jpeg` work out of the box; for `.png`, change the constant to `image/gift/gift.png`.
